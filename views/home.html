Welcome to the showcase of the official API plugin for rable.js!

rable.js-api is written by the same developer that made rable.js, and thereby the official api plugin.
The plugin adds easy management of APIs, url, and everything surrounding the creation and developement of an API architechture.

For this showcase we have set up two APIs, example and date. Normally, unless programmically intergrated by the developer like now, it's not possible for a client to list Actions nor APIs. In this example we chose to do so, so that you can try out the API.

For further information about the mechanics of how the Library works, please visit the docs (also publicly available here).
